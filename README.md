# Purgatory (Android)

A minimal, shared to-do list app backed by Google Sheets.

## Summary

- Shared Google Sheets backend (tab name: `Purgatory`)
- Two owners with color-coded tasks
- Current/Completed/All views
- Optional daily reminder

## Build & Run

1. Open the project in Android Studio.
2. Let Gradle sync.
3. Build and run on a device (Android 8.0+, target 34).

## Setup

1. Sign in with Google.
2. Paste the Spreadsheet ID in Settings.
3. Ensure the sheet has columns: `Details, Importance, Owner, Status, Date`.
